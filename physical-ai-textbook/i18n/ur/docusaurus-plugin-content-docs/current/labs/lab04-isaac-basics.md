# لیب 4: Isaac Sim بنیادیات

اس لیب میں آپ NVIDIA Isaac Sim کے ساتھ کام کرنا سیکھیں گے۔

## مقاصد

- Isaac Sim انسٹال اور لانچ کریں
- روبوٹ امپورٹ کریں
- سمولیشن چلائیں

## Isaac Sim لانچ کریں

```bash
# Isaac Sim شروع کریں
./isaac-sim.sh
```

## روبوٹ امپورٹ کریں

1. File > Import > URDF
2. اپنا روبوٹ URDF منتخب کریں
3. Import کلک کریں

## سمولیشن چلائیں

- Play بٹن دبائیں
- روبوٹ کو حرکت کرتے دیکھیں

## خلاصہ

آپ نے Isaac Sim کی بنیادیات سیکھیں۔
