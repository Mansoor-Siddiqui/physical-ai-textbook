# NVIDIA Isaac Sim

<div className="learning-objectives">

## سیکھنے کے مقاصد

اس باب کے اختتام تک، آپ قابل ہو جائیں گے:

- Isaac Sim کی آرکیٹیکچر اور صلاحیتوں کو سمجھیں
- Isaac Sim ماحول سیٹ اپ اور نیویگیٹ کریں
- Isaac Sim میں روبوٹ ماڈلز درآمد اور کنفیگر کریں
- AI ماڈلز کے لیے مصنوعی ٹریننگ ڈیٹا تیار کریں
- روبوٹ ڈویلپمنٹ کے لیے ڈیجیٹل ٹوئن ورک فلوز بنائیں
- Isaac Sim کو ROS2 کے ساتھ انٹیگریٹ کریں

</div>

## Isaac Sim کیا ہے؟

**NVIDIA Isaac Sim** NVIDIA Omniverse پر بنایا گیا ایک روبوٹکس سمولیشن پلیٹ فارم ہے۔ یہ فوٹو ریئلسٹک رینڈرنگ، درست فزکس سمولیشن، اور AI پر مبنی روبوٹس کو ڈویلپ، ٹیسٹ، اور ٹرین کرنے کے لیے طاقتور ٹولز فراہم کرتا ہے۔

### اہم خصوصیات

| خصوصیت | تفصیل |
|--------|-------|
| **فوٹو ریئلسٹک رینڈرنگ** | حقیقت پسندانہ بصری اور سینسر سمولیشن کے لیے RTX رے ٹریسنگ |
| **PhysX 5 فزکس** | درست rigid body، آرٹیکولیشن، اور سافٹ باڈی سمولیشن |
| **مصنوعی ڈیٹا جنریشن** | ٹریننگ ڈیٹا کے لیے خودکار گراؤنڈ ٹروتھ لیبلنگ |
| **ڈومین رینڈمائزیشن** | مضبوط sim-to-real ٹرانسفر کے لیے ماحول کو متغیر کریں |
| **ROS2 انٹیگریشن** | ROS2 ایکو سسٹم کے لیے نیٹو بریجز |
| **Python API** | آٹومیشن اور ٹریننگ کے لیے مکمل سکرپٹیبلٹی |
| **کلاؤڈ اسکیلیبلٹی** | متوازی طور پر ہزاروں سمولیشنز چلائیں |

## مصنوعی ڈیٹا جنریشن

Isaac Sim کا **Replicator** کامل گراؤنڈ ٹروتھ لیبلز کے ساتھ خودکار مصنوعی ڈیٹا جنریشن کو فعال کرتا ہے۔

### ڈومین رینڈمائزیشن

ڈومین رینڈمائزیشن sim-to-real ٹرانسفر کو بہتر بنانے کے لیے سین کے پہلوؤں کو متغیر کرتی ہے:
- آبجیکٹ پوزیشنز
- لائٹنگ کنڈیشنز
- مٹیریلز اور ٹیکسچرز
- کیمرہ اینگلز

## ROS2 انٹیگریشن

### Isaac Sim ROS2 بریج

Isaac Sim ROS2 کے ساتھ بغیر رکاوٹ انٹیگریشن فراہم کرتا ہے:

- کیمرہ پبلشرز
- LiDAR پبلشرز
- جوائنٹ اسٹیٹ پبلشرز
- ٹوئسٹ سبسکرائبرز

## ہیڈ لیس ٹریننگ

رینفورسمنٹ لرننگ اور بڑے پیمانے پر ٹریننگ کے لیے:

```python
# headless_training.py
from omni.isaac.kit import SimulationApp

# ہیڈ لیس موڈ میں لانچ کریں
config = {
    "headless": True,
    "width": 128,
    "height": 128,
}
simulation_app = SimulationApp(config)
```

## خلاصہ

- **Isaac Sim** RTX رینڈرنگ کے ساتھ فوٹو ریئلسٹک سمولیشن فراہم کرتا ہے
- **URDF درآمد** ROS روبوٹس کو Isaac Sim میں لاتا ہے
- **سینسر سمولیشن** میں حقیقت پسندانہ شور کے ساتھ کیمرے، LiDAR، اور IMU شامل ہیں
- **Replicator** ڈومین رینڈمائزیشن کے ساتھ مصنوعی ڈیٹا جنریشن کو خودکار کرتا ہے
- **ROS2 بریج** ROS ایکو سسٹم کے ساتھ بغیر رکاوٹ انٹیگریشن کو فعال کرتا ہے
- **ہیڈ لیس موڈ** بڑے پیمانے پر متوازی ٹریننگ کو فعال کرتا ہے

## اگلے اقدامات

اگلے باب میں، ہم ویژن-لینگویج-ایکشن (VLA) ماڈلز کی کھوج کریں گے جو Isaac Sim کی مصنوعی ڈیٹا صلاحیتوں کا استعمال کرتے ہوئے تربیت دی جا سکتی ہیں۔
